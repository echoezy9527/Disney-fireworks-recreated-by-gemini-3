## ✨ 项目介绍 

这是一个基于手势控制的 **固定迪士尼城堡背景 + 粒子烟花** 构成的互动迪士尼烟花秀，包含彗星烟花、球形烟花和垂柳烟花类型以及不同烟花组合编排。

## ✋ 手势控制说明 

请确保摄像头权限已开启，在摄像头前进行以下手势操作：

| 手势  | 动作描述 |
| :--- | :--- | 
| **手指 5 4 3 2 1** |推进章节1-5|
| **✊ 握拳** |  ending态 |

## 🛠️ 如何使用 

1. 下载本仓库中的 `.html` 文件。
2. 直接使用浏览器（推荐 Chrome）打开该文件。
3. 允许浏览器访问摄像头权限。
4. 开始挥动你的双手！
